# АнтиПрокрастинация — Web

Готовый статический сайт (HTML/CSS/JS), который можно открыть локально и опубликовать на GitHub Pages / Netlify / Vercel.

## Как запустить локально

```bash
python3 -m http.server 4173
```

Открой: `http://localhost:4173`

## Что уже работает

- План задач: добавить, отметить выполненной, удалить.
- Pomodoro: режимы 25/5/15, старт/пауза/сброс.
- Трекер привычек: добавить, отметить, удалить.
- Дашборд: задачи/прогресс/фокус-сессии.
- Сохранение в `localStorage` (данные не пропадают после перезагрузки страницы).

## Публикация на GitHub Pages

1. Создай репозиторий на GitHub и загрузи файлы:
   - `index.html`
   - `styles.css`
   - `app.js`
2. В GitHub открой **Settings → Pages**.
3. В блоке **Build and deployment** выбери:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main` и папка `/ (root)`
4. Сохрани и подожди 1–2 минуты.
5. Ссылка на сайт появится в этом же разделе.

## Публикация на Netlify (самый быстрый способ)

1. Перейди на https://app.netlify.com/drop
2. Перетащи папку проекта в окно.
3. Получишь публичную ссылку сразу.
